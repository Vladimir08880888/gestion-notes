# Instructions pour la capture d'écran

Pour ajouter la capture d'écran dans le README :

1. Prenez une capture d'écran de l'application en cours d'exécution
2. Sauvegardez-la dans ce dossier avec le nom : `interface-principale.png`

Sur Mac :
- Cmd + Shift + 4 : pour capturer une zone spécifique
- Cmd + Shift + 3 : pour capturer tout l'écran

Sur Windows :
- Windows + Shift + S : pour l'outil de capture
- Print Screen : pour capturer tout l'écran

La capture d'écran sera automatiquement affichée dans le README.md
